# UrbanNest_COMP313004/ Final project 
Technologies Used

Backend: Node.js,Express.js

DataBase: MongoDB

Frontend: React.js

Prerequisites:
Node.js (latest LTS version recommended),

Clone the repository and run it locally:

git clone https://github.com/khaliddawd/UrbanNest_COMP313004.git
